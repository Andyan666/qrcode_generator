# 二维码生成器

一个功能丰富的 Python tkinter 二维码生成器应用，具有现代化界面设计。

## 功能特性

### 基础功能
- 输入文本或网址生成二维码
- 复制文本到剪贴板 / 从剪贴板粘贴
- 容错级别选择（低/中/高）
- 多格式保存（PNG、JPG、SVG、PDF）

### 样式定制
- **前景色选择**：自定义二维码颜色，色块实时预览
- **背景色选择**：自定义背景颜色，色块实时预览
- **Logo 嵌入**：在二维码中央嵌入 Logo 图片（自动缩放、白色背景边框）
- **边框宽度**：输入框设置边距（0-10），超出范围自动限制
- **输出尺寸**：输入框设置分辨率（100-800像素），适合不同用途

### 高级功能
- **动态二维码**：监测 URL 内容变化，每 5 秒自动重新生成
- **批量生成**：从 TXT/CSV 文件批量生成二维码，自动命名保存
- **Wi-Fi 二维码**：生成可直接扫码连 Wi-Fi 的二维码

## 界面设计

- 现代化浅灰色背景 (#f5f5f5)
- 分组布局（输入内容、样式设置、二维码预览）
- 彩色功能按钮：
  - 🟢 生成二维码（绿色）
  - 🔵 动态监测（蓝色，运行时变红色）
  - 🟠 批量生成（橙色）
  - 🟣 Wi-Fi 二维码（紫色）
- 预览区域白色背景，清晰显示二维码


## 使用说明

### 生成普通二维码
1. 在文本框输入内容
2. 设置样式参数：
   - 点击色块旁的"选择"按钮自定义颜色
   - 选择 Logo 图片（可选）
   - 输入边框宽度（0-10）
   - 输入输出尺寸（100-800）
   - 选择容错级别
3. 点击"生成二维码"
4. 选择保存格式，点击"保存图片"

### 批量生成
1. 准备 TXT 或 CSV 文件，每行一个内容
2. 点击"批量生成"按钮
3. 选择文件和保存目录
4. 点击"开始生成"，自动命名为 `qrcode_001.png`、`qrcode_002.png`...

### Wi-Fi 二维码
1. 点击"Wi-Fi二维码"按钮
2. 输入网络名称(SSID)、密码
3. 选择加密类型（WPA/WPA2、WEP、无密码）
4. 可选隐藏网络、显示密码
5. 点击"生成二维码"预览
6. 点击"保存图片"

### 动态监测
1. 输入 URL 地址（需以 http:// 或 https:// 开头）
2. 点击"启动动态监测"
3. 程序每 5 秒检查 URL 内容变化
4. 内容变化时自动重新生成二维码
5. 点击"停止动态监测"结束

## 容错级别说明

| 级别 | 容错率 | 说明 |
|------|--------|------|
| 低 (L) | 7% | 适合清晰环境 |
| 中 (M) | 15% | 默认推荐 |
| 高 (H) | 30% | 适合嵌入 Logo 或恶劣环境 |

## 参数范围

| 参数 | 范围 | 说明 |
|------|------|------|
| 边框宽度 | 0-10 | 二维码边距，超出自动限制 |
| 输出尺寸 | 100-800 | 像素，小尺寸适合屏幕，大尺寸适合打印 |

## 注意事项

- 嵌入 Logo 时建议使用高容错级别
- 大尺寸输出适合打印用途
- SVG 格式为矢量图，适合进一步编辑
- PDF 格式自动处理透明背景
- 动态监测仅支持 HTTP/HTTPS URL
- 输入参数超出范围会自动调整到有效值
